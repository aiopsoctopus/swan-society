# Swan Society — Brand Kit

A reference sheet for the visual identity, plus ready-to-paste prompts for Gemini / ChatGPT to generate logos and collateral. Pair this with `Design Inspiration and Lovable Prompts.md` (the app UI system) — this doc is for everything *outside* the app: logo files, social assets, print collateral, merch.

---

## 1. Brand essence

- **Name**: Swan Society
- **Tagline options**: "A book club for curious souls" / "A book club for the curious & the well-read" / "Membership by invitation"
- **Vibe**: 1920s Gatsby-era literary salon — exclusive, elegant, a little glamorous. Warm jewel tones, gold line work, deco geometry.
- **Audience**: busy women who want their book club to feel special without the admin burden.

## 2. Color palette

| Name | Hex | Use |
|---|---|---|
| Emerald | `#0C2A24` / `#0E1F1C` | Primary dark surface, backgrounds |
| Garnet | `#3D0F1E` / `#4A1B28` | Secondary accent, depth |
| Midnight navy | `#101935` / `#1B2A4A` | Secondary dark surface |
| Onyx | `#1C1C1C` | Text, high-contrast surfaces |
| Champagne gold | `#D4AF6A` / `#C9A86A` | Primary accent — line work, foil, CTAs |
| Ivory | `#F4EFE3` | Cards, negative space, light backgrounds |

## 3. Typography

- **Headlines**: deco-style serif, set in spaced-out capital letters (e.g. "SWAN SOCIETY")
- **Body / UI**: clean modern sans-serif
- Avoid: script fonts, anything too "costume party" — the deco should feel refined, not themed.

## 4. Logo system

- **Primary mark (locked)**: a solid champagne-gold filled swan, neck curved into an S-shape, wings layered in flowing feather strokes, sitting atop an open book rendered in the same solid gold. The swan + book sit inside a diamond-shaped art deco frame (not a plain circle) built from sunburst rays, with small fan/chevron ornaments at the four points of the diamond. Wordmark "SWAN SOCIETY" in spaced deco serif caps below, with tagline "A book club for curious souls" in clean sans-serif underneath. All gold on deep emerald background.
- **Style note**: solid fill (not line art) — this is what makes the swan instantly legible at a glance and gives the mark a confident, jewelry-like quality.
- **Variants needed**: full lockup (mark + wordmark, locked above), mark only (icon/favicon — diamond frame + swan/book, no text), horizontal lockup (mark + wordmark side-by-side for headers), monochrome versions (all-gold on dark, all-onyx on ivory, all-ivory on dark)

## 5. Pattern language

Deco motifs — sunburst/fan shapes, chevrons, thin gold dividers — used sparingly as accents (section breaks, card borders, packaging edges). Treat like jewelry, not wallpaper.

## 7. Usage notes

- **Consistency tip**: the primary logo is locked (solid gold swan + book in a diamond deco frame, on emerald). Save that image and attach it as a reference in prompts 6.2–6.8 wherever the tool supports image input/editing — this is the single biggest lever for keeping variants consistent rather than drifting stylistically.
- **Watch for text glitches**: AI image generators frequently garble "SOCIETY" (e.g. overlapping/doubled letters like "SOCIIETY"). Check wordmark legibility on every generation, and regenerate or retype the wordmark separately in a design tool (Canva/Figma) if it's off — don't settle for a glitchy version.
- **File formats**: AI image generators output PNG/JPG, not vector. For a scalable logo (favicon, print at large sizes), consider vectorizing a chosen result afterward (e.g. via a vector-trace tool) — flag this if you need crisp edges at large sizes.
- **Don't overuse deco patterns**: across collateral, keep sunburst/chevron motifs to one accent per piece. The brand should read as elegant, not "themed party."
